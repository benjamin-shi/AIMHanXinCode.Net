Parameters
data
Type: SystemByte
Data will be encoded into Han Xin Code.
version
Type: SystemInt32

 User-chosen version of Han Xin Code.
 
The returnable value of this parameter will be the final version chosen by encoding process. This will be happen, if user-chosen version of Han Xin Code cannot contain all of data.
If the version of Han Xin Code could not be easily considered, please set to 0.
error_correction_level
Type: SystemInt32

 User-chosen error correction level of Han Xin Code.
 
1(L1): 8%
2(L2):15%
3(L3):23%
4(L4):30%
quiet_zone_size (Optional)
Type: SystemInt32
User-chosen quiet zone in modules of Han Xin Code.
Return Value
Type: Byte

 Result of encoding process.
 
If encoding process successes, the result will be a two dimensional Byte array.
If encoding process fails, the result will be Nothing.
